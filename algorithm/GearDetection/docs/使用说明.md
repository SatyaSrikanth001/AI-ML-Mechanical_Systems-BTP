# 齿轮缺陷检测平台使用说明

## 系统概述

齿轮缺陷检测平台是一个基于深度学习的齿轮缺陷自动识别系统，可以检测齿轮表面的点蚀、剥落、压伤和擦伤等常见缺陷。系统采用YOLO模型进行目标检测，通过Web界面提供友好的用户交互体验。

## 系统架构

系统由以下部分组成：

1. **前端界面**：基于HTML、CSS和JavaScript (React) 构建的用户界面
2. **后端API**：基于Flask的RESTful API服务
3. **深度学习模型**：基于YOLO的齿轮缺陷检测模型

## 快速启动

### 环境准备

1. 确保已安装Python 3.7+
2. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```
3. 将训练好的模型文件（.pt格式）放入`model`目录，并命名为`best.pt`

### 启动服务

1. 启动后端API服务：
   ```
   python api/app.py
   ```
2. 在浏览器中打开前端页面：
   ```
   http://localhost:5000
   ```
   或直接打开`home.html`文件

## 使用流程

1. **上传图像**：点击上传区域选择齿轮图像（支持JPG、PNG格式）
2. **调整参数**：根据需要调整检测置信度阈值（默认0.25）
3. **开始检测**：点击"开始检测"按钮进行缺陷识别
4. **查看结果**：系统将在右侧显示检测结果，包括缺陷位置、类型和置信度
5. **下载结果**：可以点击"下载结果"按钮保存带有检测框的结果图像

## 缺陷类型说明

系统可以识别以下四种齿轮缺陷：

| 缺陷类型 | 英文名称 | 描述 |
|---------|---------|------|
| 点蚀 | Pitting | 表面点状腐蚀或磨损 |
| 剥落 | Spalling | 表面材料剥落脱落 |
| 压伤 | Indentation | 受压产生的凹陷变形 |
| 擦伤 | Scratching | 表面划痕或擦痕 |

## 常见问题

1. **API连接失败**
   - 检查后端服务是否正常运行
   - 确认API地址配置是否正确
   - 检查网络连接是否正常

2. **检测结果不准确**
   - 尝试调整置信度阈值
   - 确保图像质量良好，光照充足
   - 检查模型文件是否正确加载

3. **图像上传失败**
   - 确认图像格式是否支持（JPG、PNG）
   - 检查图像大小是否超过限制（16MB）
